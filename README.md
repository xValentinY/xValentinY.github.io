# AnimeLab - Sitio educativo (HTML, CSS, JavaScript)

## Descripción
Proyecto estático para uso en el aula. Contiene:
- `index.html` — estructura y contenido.
- `css/style.css` — estilos con variables, layout responsive y media queries.
- `js/main.js` — interactividad: tema claro/oscuro, lightbox, validación de formulario.

## Cómo usar
1. Descarga y descomprime `anime_edu_site.zip`.
2. Abre la carpeta en Visual Studio Code (`File -> Open Folder...`).
3. Abre `index.html` en el navegador (doble clic) o usa la extensión Live Server.
4. Edita archivos en `css/`, `js/` o `assets/` para tus clases.

## Ideas de ejercicios para alumnos
- Modifica las variables en `:root` para cambiar la paleta.
- Reemplaza los SVG en `assets/` por imágenes reales.
- Cambia el comportamiento del botón de perfil para mostrar un modal en lugar de `alert`.
- Agrega animaciones CSS a las tarjetas o transiciones JS al lightbox.
